# cabledrive   
Programy v MPLABX s kompilatorem XC8 pro komunikaci mezi deskami ovladace a mezikusu pro DC  pohon s kontrolery PIC16f876  
DC_OVL.X: Program pro tlacitkovy ovladac s joystikem  
DC_MKUS.X: Program pro komunikaci s ovladacem po RS485, ovladani doplnenych zarizeni a signaly pro DC pohon
